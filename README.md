# 🌍 PlanMyWay.com — Your Personal Travel Planner

PlanMyWay.com is a responsive and interactive travel and tour website built with **HTML**, **CSS**, and **JavaScript**. Designed to provide users with an engaging and smooth travel exploration experience, it features curated destinations, adventure ideas, and personalized tour packages across India.

## 📸 Features

- ✅ **Responsive Design** — Seamlessly adapts to mobile, tablet, and desktop screens.
- ✅ **Smooth Navigation** — Mobile menu toggle, scroll-based navbar effects.
- ✅ **Search Overlay** — Animated full-screen search form.
